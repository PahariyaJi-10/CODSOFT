# 🌟 QuoteVerse - Quote of the Day App

QuoteVerse is a modern Android application built using Kotlin and Jetpack Compose that delivers daily inspirational quotes with a beautiful and smooth user experience.

# 📱 Features

✅ Modern Premium UI/UX  
✅ Random Inspirational Quotes  
✅ Smooth Quote Animations  
✅ Share Quotes to Other Apps  
✅ Favorite Quotes System ❤️  
✅ Separate Favorites Screen  
✅ Beautiful Gradient Backgrounds  
✅ Glassmorphism Style Cards  
✅ Navigation Between Screens  
✅ Material 3 Design  
✅ Responsive Layout  

# 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Navigation Compose
- Android Studio


# 📸 Screens Included

- Home Screen
- Favorite Quotes Screen


# ✨ UI Highlights

- Animated quote transitions
- Rounded modern cards
- Smooth layouts and spacing
- Elegant typography
- Dark premium color palette

# ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/PahariyaJi-10/CODSOFT.git
```

2. Open the project in Android Studio

3. Sync Gradle

4. Run the application on:
   - Android Emulator
   - OR Physical Android Device
